1. Cas d'utilisation. 

Le scrapping permet de recupérer des informations directement sur des sites web. En extrayant le code source d'une page web, nous pouvons recupérer les informations voulues. 


2. Outils. 

- Python 
- BeautifulSoup 
- Selenium 
- Request 
- Un navigateur web et les outils de dev web 


3. Processus en python. 

- On requete une page.
- On regarde sa structure depuis les outils de dev de d'un navigateur. 
- On requete à l'aide de l'une des librairies (Selenium ou BeautifulSoup) pour selectionner les éléments que l'on veut. 


4. Difference BeautifulSoup et Selenium.

Selenium est conçu pour automatiser les interactions avec les pages web, tandis que BeautifulSoup est principalement utilisé pour analyser et extraire des données du HTML statique.
